# 🚀 AI Meeting Intelligence System

An **AI-powered meeting analysis platform** that automatically transcribes meeting audio and generates intelligent insights including summaries, action items, topics, sentiment, and analytics.

The system uses **speech recognition, natural language processing, and generative AI** to convert meeting recordings into structured business insights.

---

# 🌐 Live Demo

🔗 https://ai-meeting-intelligence-system.streamlit.app

---

# 🧠 Features

• 🎤 **Automatic Speech-to-Text** using Whisper
• 📄 **AI Meeting Summarization**
• ✅ **Action Item Extraction**
• 🧩 **Topic Detection**
• 🔑 **Keyword Extraction**
• 😊 **Meeting Sentiment Analysis**
• 📊 **Interactive Analytics Dashboard**
• 📥 **Download Meeting Transcript**

---

# ⚙️ How It Works

```
Meeting Audio
      │
      ▼
Speech-to-Text (Whisper)
      │
      ▼
Transcript Processing
      │
      ▼
LLM Analysis (Groq)
      │
      ▼
Summary | Action Items | Topics
      │
      ▼
Analytics Dashboard
```
